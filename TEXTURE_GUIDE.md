# 纹理资源说明

## 需要创建的纹理文件

本模组需要以下纹理文件（16x16 PNG 格式）：

### 物品纹理
- `src/main/resources/assets/craft_enchanted_book/textures/item/enchanted_quill.png`
  - 特殊羽毛笔纹理
  - 建议：绘制一个发光的羽毛笔图案

- `src/main/resources/assets/craft_enchanted_book/textures/item/guide_book.png`
  - 说明书纹理
  - 建议：绘制一本带有魔法符号的书籍

## 纹理制作提示

1. **尺寸**: 16x16 像素（标准物品纹理）
2. **格式**: PNG（支持透明度）
3. **风格**: 与 Minecraft 原版风格保持一致
4. **颜色**: 使用明亮的魔法色调（紫色、金色等）

## 临时方案

在正式纹理创建之前，可以使用以下方法生成临时纹理：

1. 复制 Minecraft 原版的羽毛笔纹理作为基础
2. 使用图像编辑软件（如 GIMP、Photoshop）进行修改
3. 添加发光效果或魔法粒子效果

## 纹理位置

所有纹理文件应放置在：
```
src/main/resources/assets/craft_enchanted_book/textures/item/
```
